# 🌐 Бесплатный хостинг бота 24/7

## 🎯 Лучшие бесплатные варианты

### 1️⃣ Render.com (РЕКОМЕНДУЕТСЯ)
**✅ Плюсы:** Простой, надёжный, 750 часов/месяц бесплатно  
**❌ Минусы:** Засыпает после 15 минут неактивности

**Инструкция:**

1. **Зарегистрируйся на Render.com**
   - Перейди на https://render.com
   - Нажми "Get Started"
   - Войди через GitHub

2. **Создай GitHub репозиторий**
   - Создай новый репозиторий на GitHub
   - Загрузи все файлы бота
   - Добавь файл `requirements.txt`

3. **Создай Web Service на Render**
   - Нажми "New +" → "Web Service"
   - Подключи свой GitHub репозиторий
   - Настройки:
     - Name: `telegram-antibot`
     - Environment: `Python 3`
     - Build Command: `pip install -r requirements.txt`
     - Start Command: `python bot.py`
   - Нажми "Create Web Service"

4. **Готово!** Бот запустится автоматически

**Чтобы бот не засыпал:**
- Используй сервис UptimeRobot (https://uptimerobot.com)
- Создай HTTP(s) монитор на URL твоего Render сервиса
- Проверка каждые 5 минут

### 7️⃣ Oracle Cloud (ЛУЧШИЙ для 24/7)
**✅ Плюсы:** Бесплатно навсегда, мощный, 24/7  
**❌ Минусы:** Сложная настройка

**Инструкция:**

1. **Зарегистрируйся**
   - https://cloud.oracle.com
   - Создай бесплатный аккаунт (требует карту, но не списывает)

2. **Создай VM Instance**
   - Compute → Instances → Create Instance
   - Shape: VM.Standard.E2.1.Micro (бесплатный)
   - Image: Ubuntu 20.04
   - Создай SSH ключ

3. **Подключись по SSH**
   ```bash
   ssh ubuntu@<IP-адрес>
   ```

4. **Установи Python и зависимости**
   ```bash
   sudo apt update
   sudo apt install python3 python3-pip
   pip3 install python-telegram-bot
   ```

5. **Загрузи файлы бота**
   ```bash
   # Используй scp или git
   git clone <твой-репозиторий>
   cd <папка-бота>
   ```

6. **Запусти бота как сервис**
   ```bash
   # Создай systemd сервис
   sudo nano /etc/systemd/system/antibot.service
   ```

   Вставь:
   ```ini
   [Unit]
   Description=Telegram Anti-Bot
   After=network.target

   [Service]
   Type=simple
   User=ubuntu
   WorkingDirectory=/home/ubuntu/bot
   ExecStart=/usr/bin/python3 /home/ubuntu/bot/bot.py
   Restart=always

   [Install]
   WantedBy=multi-user.target
   ```

   Запусти:
   ```bash
   sudo systemctl enable antibot
   sudo systemctl start antibot
   sudo systemctl status antibot
   ```

7. **Готово!** Бот работает 24/7
