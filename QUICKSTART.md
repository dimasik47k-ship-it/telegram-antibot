# ⚡ Быстрый старт

## За 5 минут до запуска!

### Шаг 1: Установи Python (если нет)

**Windows:** https://www.python.org/downloads/
**Linux:** `sudo apt install python3 python3-pip`
**macOS:** `brew install python3`

### Шаг 2: Скачай бота

Скачай все файлы в одну папку.

### Шаг 3: Установи зависимости

```bash
pip install python-telegram-bot
```

Или:

```bash
pip install -r requirements.txt
```

### Шаг 4: Запусти бота

**Windows:**
```bash
python bot.py
```
Или двойной клик на `start.bat`

**Linux/macOS:**
```bash
python3 bot.py
```
Или `./start.sh`

### Шаг 5: Добавь в группу

1. Найди бота в Telegram
2. Добавь в группу
3. Дай права администратора:
   - ✅ Удаление сообщений
   - ✅ Бан пользователей
   - ✅ Ограничение пользователей

### Готово! 🎉

Бот автоматически начнёт проверять новых участников.

## Основные команды

```
/start - Начать
/help - Помощь
/stats - Статистика
/verify - Пройти проверку
```

## Для админов

```
/ban @user - Забанить
/kick @user - Кикнуть
/mute @user 30 - Замутить на 30 мин
/settings - Настройки
```

## Настройка (опционально)

Открой `config.py` и измени:

```python
# Количество попыток
'max_attempts': 5

# Время на проверку (секунды)
'timeout': 300

# Типы проверок
'enabled_types': ['emoji', 'math', 'logic']
```

## Проблемы?

1. Проверь, что Python установлен: `python --version`
2. Проверь, что библиотека установлена: `pip list | grep telegram`
3. Проверь логи: `bot.log`
4. Перезапусти бота
