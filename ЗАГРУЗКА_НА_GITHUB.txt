📋 ШАГ 1: СОЗДАЙ РЕПОЗИТОРИЙ (1 минута)

1. Открой в браузере:
   https://github.com/new

2. Заполни:
   Repository name: telegram-antibot
   Description: Telegram bot для защиты от ботов
   Public или Private: любой
   
   ⚠️ НЕ СТАВЬ галочки:
   ❌ Add a README file
   ❌ Add .gitignore
   ❌ Choose a license

3. Нажми зелёную кнопку "Create repository"

═══════════════════════════════════════════════════════════════

📋 ШАГ 2: ЗАГРУЗИ ZIP-АРХИВ (1 минута)

1. На странице нового репозитория найди текст:
   "uploading an existing file"
   
   Или просто нажми:
   "Add file" → "Upload files"

2. Перетащи файл telegram-antibot.zip в окно браузера
   
   ИЛИ
   
   Нажми "choose your files" и выбери telegram-antibot.zip

3. Подожди, пока файл загрузится (несколько секунд)

4. Внизу страницы нажми зелёную кнопку:
   "Commit changes"

═══════════════════════════════════════════════════════════════

📋 ШАГ 3: РАСПАКУЙ АРХИВ НА GITHUB (30 секунд)

1. На странице репозитория нажми на файл:
   telegram-antibot.zip

2. Нажми кнопку "Download" (скачать)
   
   ⚠️ НЕТ! Не скачивай! Вместо этого:

3. Удали ZIP и загрузи файлы по отдельности:
   
   a) Распакуй telegram-antibot.zip на своём компьютере
   
   b) Вернись на GitHub
   
   c) Нажми "Add file" → "Upload files"
   
   d) Перетащи ВСЕ распакованные файлы
   
   e) Нажми "Commit changes"

📋 ШАГ 4: ЗАРЕГИСТРИРУЙСЯ НА RENDER (2 минуты)

1. Открой: https://render.com

2. Нажми "Get Started" или "Sign Up"

3. Выбери "Sign up with GitHub"

4. Войди в GitHub (если попросит)

5. Разреши Render доступ к репозиториям

═══════════════════════════════════════════════════════════════

📋 ШАГ 5: СОЗДАЙ WEB SERVICE (2 минуты)

1. На главной Render нажми "New +"

2. Выбери "Web Service"

3. Найди репозиторий "telegram-antibot"
   
   Если не видишь:
   - Нажми "Configure account"
   - Дай доступ к репозиторию

4. Нажми "Connect"

5. Заполни (большинство заполнится автоматически):
   
   Name: telegram-antibot
   Region: любой (Oregon рекомендуется)
   Branch: main
   Environment: Python 3
   
   Build Command: pip install -r requirements.txt
   Start Command: python bot.py
   
   Plan: Free

6. Нажми "Create Web Service"

7. Подожди 2-3 минуты, пока бот запустится

═══════════════════════════════════════════════════════════════

📋 ШАГ 6: ПРОВЕРЬ РАБОТУ (30 секунд)

1. Когда в логах увидишь:
   "Бот запущен!"
   "Application started"

2. Открой Telegram

3. Найди своего бота

4. Отправь /start

5. Бот ответил? ✅ РАБОТАЕТ!

═══════════════════════════════════════════════════════════════

📋 ШАГ 7: НАСТРОЙ UPTIMEROBOT (2 минуты)

⚠️ ОБЯЗАТЕЛЬНО! Иначе бот будет засыпать через 15 минут!

1. Открой: https://uptimerobot.com

2. Нажми "Sign Up" (бесплатно)

3. После регистрации нажми "+ Add New Monitor"

4. Заполни:
   
   Monitor Type: HTTP(s)
   Friendly Name: Telegram Bot
   URL: (скопируй URL с Render, например:
        https://telegram-antibot.onrender.com)
   Monitoring Interval: 5 minutes

5. Нажми "Create Monitor"

6. Готово! Бот НЕ БУДЕТ засыпать!

❓ ПРОБЛЕМЫ?

Не могу загрузить файлы на GitHub:
- Попробуй другой браузер
- Проверь интернет
- Загружай файлы по одному

Render не видит репозиторий:
- Нажми "Configure account"
- Дай доступ к репозиториям
- Обнови страницу

Бот не запускается:
- Проверь логи на Render
- Убедись, что все файлы загружены
- Проверь requirements.txt

Бот не отвечает:
- Проверь токен в config.py
- Убедись, что бот запущен (зелёный статус)
- Проверь логи
