🎯 ШАГ 2: СОЗДАЙ BACKGROUND WORKER НА RENDER

1. Зайди на https://dashboard.render.com

2. Нажми "New +" → "Background Worker"

3. Подключи репозиторий:
   • Connect GitHub repository
   • Выбери: dimasik47k-ship-it/telegram-antibot

4. Настрой сервис:
   
   Name: telegram-antibot-worker
   
   Region: Frankfurt (EU Central)
   
   Branch: main
   
   Runtime: Python 3
   
   Build Command: pip install -r requirements.txt
   
   Start Command: python bot.py
   
   Instance Type: Free

5. Нажми "Create Background Worker"
