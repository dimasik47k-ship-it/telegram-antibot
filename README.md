# 🤖 Telegram Anti-Bot Protection

Мощный Telegram бот для защиты группы от ботов и ИИ с расширенным функционалом.

## 🎯 Основные возможности

### ✅ Система верификации
- **10 типов проверок** с динамической генерацией
- **5 попыток** на прохождение
- **5 минут** на выполнение
- Автоматическое ограничение прав до прохождения
- Уведомления в личные сообщения

### 📊 Типы проверок

1. **Эмодзи** - найди правильный эмодзи из категории
2. **Математика** - реши простой пример (+, -, *)
3. **Последовательность** - продолжи числовой ряд
4. **Цвета** - выбери названный цвет
5. **Паттерн** - найди повторяющийся элемент
6. **Логика** - ответь на простой вопрос
7. **Сравнение** - сравни два числа
8. **Слова** - найди животное по названию
9. **Счёт** - посчитай количество эмодзи
10. **Лишнее** - найди элемент из другой категории

### 🛡️ Защита

- Автоматическая проверка новых участников
- Ограничение прав до верификации
- Автоматический кик при провале
- Защита от повторного входа забаненных
- Белый и чёрный списки
- Система предупреждений

### 👑 Команды для администраторов

```
/ban @username - Забанить пользователя
/unban @username - Разбанить пользователя
/kick @username - Кикнуть пользователя
/mute @username [минуты] - Замутить пользователя
/unmute @username - Размутить пользователя
/warn @username [причина] - Предупредить пользователя
/whitelist @username - Добавить в белый список
/blacklist @username - Добавить в чёрный список
/settings - Настройки группы
```

### 📱 Команды для пользователей

```
/start - Начать работу с ботом
/help - Показать справку
/verify - Пройти проверку вручную
/status - Проверить статус верификации
/stats - Статистика бота
```

## 🚀 Установка и запуск

### Требования

- Python 3.8+
- pip

### Установка

1. Клонируй репозиторий или скачай файлы

2. Установи зависимости:
```bash
pip install -r requirements.txt
```

3. Запусти бота:
```bash
python bot.py
```

## ⚙️ Настройка

Токен бота уже прописан в коде:
```python
BOT_TOKEN = "7681253398:AAGAr7kug1Vd0o1yNsk6yJj6B8R3Xfnf8Hk"
```

### Настройки по умолчанию

- **Время на проверку**: 5 минут (300 секунд)
- **Количество попыток**: 5
- **Интервал очистки**: 10 минут (600 секунд)

Изменить можно в константах в начале файла `bot.py`:
```python
MAX_ATTEMPTS = 5
VERIFICATION_TIMEOUT = 300
CLEANUP_INTERVAL = 600
```

Бот автоматически:
