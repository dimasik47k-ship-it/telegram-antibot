# 🤖 Telegram Anti-Bot Protection

Мощный бот для защиты Telegram групп от ботов и ИИ.

## ⚡ Быстрый старт

```bash
# 1. Установи библиотеку
python -m pip install python-telegram-bot

# 2. Запусти бота
python bot.py

# 3. Добавь в группу и дай права администратора
```

## 🎯 Возможности

- ✅ 10 типов проверок (эмодзи, математика, логика и др.)
- ✅ Автоматическая верификация новых участников
- ✅ Модерация (бан, кик, мут)
- ✅ Статистика и аналитика
- ✅ Полная автономность (без внешних сервисов)

## 📱 Команды

**Для всех:**
- `/start` - Начать работу
- `/help` - Помощь
- `/verify` - Пройти проверку
- `/stats` - Статистика

**Для админов:**
- `/ban @user` - Забанить
- `/kick @user` - Кикнуть
- `/mute @user 30` - Замутить на 30 мин
- `/settings` - Настройки

## ⚙️ Настройка

Открой `config.py` и измени параметры:

```python
'max_attempts': 5      # Количество попыток
'timeout': 300         # Время на проверку (секунды)
'enabled_types': [...]  # Типы проверок
```
