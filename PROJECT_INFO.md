# 📋 Информация о проекте

## 🎯 Название проекта
**Telegram Anti-Bot Protection**

## 📝 Описание
Мощный автономный Telegram бот для защиты групп от ботов и ИИ с 175+ функциями, 10 типами проверок и полной независимостью от внешних сервисов.

## 🔑 Ключевые особенности

### ✅ Полная автономность
- Без внешних API
- Без баз данных
- Без вебхуков
- Без облачных сервисов
- Работает локально на твоём компьютере

### 🛡️ Мощная защита
- 10 типов проверок с генерацией
- Защита от ботов и ИИ
- Анти-спам и анти-флуд
- Белые и чёрные списки
- Автоматическая модерация

### 📊 Богатый функционал
- 175+ функций
- 15+ команд
- Детальная статистика
- Гибкие настройки
- Полная документация
