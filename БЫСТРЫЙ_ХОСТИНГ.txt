📋 ШАГ 1: СОЗДАЙ GITHUB РЕПОЗИТОРИЙ

1. Зайди на https://github.com
2. Нажми "New repository"
3. Название: telegram-antibot
4. Нажми "Create repository"

═══════════════════════════════════════════════════════════════

📋 ШАГ 2: ЗАГРУЗИ ФАЙЛЫ НА GITHUB

В терминале (в папке с ботом):

git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/твой-username/telegram-antibot.git
git push -u origin main

Или через веб-интерфейс GitHub:
- "Add file" → "Upload files"
- Перетащи все файлы
- "Commit changes"

═══════════════════════════════════════════════════════════════

📋 ШАГ 3: ЗАРЕГИСТРИРУЙСЯ НА RENDER.COM

1. Зайди на https://render.com
2. Нажми "Get Started"
3. Войди через GitHub
4. Разреши доступ к репозиториям

═══════════════════════════════════════════════════════════════

📋 ШАГ 4: СОЗДАЙ WEB SERVICE

1. Нажми "New +" → "Web Service"
2. Найди свой репозиторий telegram-antibot
3. Нажми "Connect"

Настройки:
- Name: telegram-antibot
- Environment: Python 3
- Build Command: pip install -r requirements.txt
- Start Command: python bot.py
- Plan: Free

4. Нажми "Create Web Service"

═══════════════════════════════════════════════════════════════

📋 ШАГ 5: ПОДОЖДИ ЗАПУСКА

Render автоматически:
✅ Установит Python
✅ Установит зависимости
✅ Запустит бота

Это займёт 2-3 минуты.

═══════════════════════════════════════════════════════════════

📋 ШАГ 6: ПРОВЕРЬ РАБОТУ

1. Открой Telegram
2. Найди своего бота
3. Отправь /start
4. Должен ответить!

═══════════════════════════════════════════════════════════════

⚠️ ВАЖНО: БОТ ЗАСЫПАЕТ ЧЕРЕЗ 15 МИНУТ

Решение - используй UptimeRobot:

1. Зайди на https://uptimerobot.com
2. Создай бесплатный аккаунт
3. "Add New Monitor"
4. Monitor Type: HTTP(s)
5. URL: твой URL с Render (например: https://telegram-antibot.onrender.com)
6. Monitoring Interval: 5 minutes
7. "Create Monitor"

Теперь бот НЕ БУДЕТ засыпать!

❓ ПРОБЛЕМЫ?

Бот не запускается:
- Проверь логи на Render
- Убедись, что все файлы загружены
- Проверь requirements.txt

Бот засыпает:
- Настрой UptimeRobot
- Проверь, что монитор активен

Бот не отвечает:
- Проверь токен в config.py
- Убедись, что бот запущен на Render
